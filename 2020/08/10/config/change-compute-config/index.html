更换电脑之后博客迁移操作

开始准备

默认提前安装好git,nodejs

拷贝必要文件

  • _config.yml
  • theme
  • source
  • scaffolds
  • package.json
  • .gitignore

安装必要环境

npm install hexo-cli -g
npm install

必要组件
sudo npm install hexo-deployer-git --save
sudo npm install hexo-generator-feed --save
sudo npm install hexo-generator-sitemap --save

git部署

将默认私钥添加至.ssh路径

git初始化一下即可
