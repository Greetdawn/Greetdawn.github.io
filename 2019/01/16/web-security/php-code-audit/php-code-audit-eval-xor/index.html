代码分析

get传入的cover变量值必须要与session值相等，这里可以使用session绕过，直接传入一个空的cover；

通过get传入code变量值，code变量字符串的长度不能超过35位;且字符串不能含有数字、字母、_、.、+等特殊字符；

最终code变量带入eval函数,这里首先传入函数getPos()；

关于怎么绕过的字符串的过滤，这里可以使用特殊字符串的异或来实现，直接上异或脚本:
